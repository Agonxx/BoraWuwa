# BoraWuwa

Central de Ressonadores — hub pessoal de referência para builds de Wuthering Waves. React + Vite + TypeScript.

Veja [`STATUS.md`](STATUS.md) para o relatório de progresso e checklist de pendências, e [`Design/README.md`](Design/README.md) para o spec de design original.

## Rodando localmente

```bash
npm install
npm run dev
```

## Scripts

```bash
npm run dev       # servidor de desenvolvimento
npm run build     # build de produção
npm run lint      # oxlint
```
